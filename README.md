# Morse Code Research

## 1. Overview of Morse Code
Morse code is a method of encoding text characters as sequences of dots and dashes. Each letter, number, and punctuation mark is represented by a unique combination of short signals `dot .` and long signals `dash -`. 
It was originally designed for use in telegraph and radio communication to transmit messages quickly and reliably over long distances.

## 2. Historical Context
Morse code was developed in the 1830s and 1840s by Samuel Morse and Alfred Vail in the United States. 
It was created to work with the electric telegraph, which was the first practical system for long-distance communication. 
Before telephones and the internet, Morse code allowed operators to send messages across wires and later via radio waves. 
During the 19th and 20th centuries, it became the standard for maritime, military, and aviation communication. 
Although it is less used today, Morse code is still taught and used by amateur radio operators.

## 3. How the Morse Code System Works
The Morse code system uses only two elements:
- **Dot `.`** : A short signal
- **Dash `-`** : A long signal, about 3 times the length of a dot

Rules:
1. The space between dots and dashes within one letter = 1 unit
2. The space between letters in a word = 3 units  
3. The space between words = 7 units
